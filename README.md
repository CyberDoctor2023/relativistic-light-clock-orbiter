## üî≠ About The Project

**Relativistic Light Clock Orbiter** is an educational visualization tool designed to demonstrate the counter-intuitive effects of Special Relativity. By simulating a light clock on a moving platform, users can intuitively grasp concepts like time dilation and length contraction.

Built with **Electron**, **React**, and **Three.js**, this application delivers high-performance 3D graphics directly to your desktop.

## üöÄ Getting Started

### Prerequisites

*   **Node.js**: v18 or higher recommended.
*   **npm** or **yarn**.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/CyberDoctor2023/relativistic-light-clock-orbiter.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run Development Server
    ```sh
    npm run dev
    ```

### Building for Production

To create the DMG installer for macOS:

```sh
npm run dist
```
The output file will be located in the `dist/` directory.
